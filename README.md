# HelloServletProject

This project demonstrates a basic Java servlet application deployed on Apache Tomcat.

## Steps Followed
1. **Downloaded and installed Apache Tomcat (10.1.52)**
2. **Created a Java Servlet** that prints "Hello, Edmond Chakar!" to the browser.
3. **Created a simple HTML page** with a link to the servlet.
4. **Deployed the servlet** on Apache Tomcat and tested it.

## Software and Versions Used
- **Apache Tomcat 10.1.52**
- **Java 24**
- **Maven 3.8.4**
- **IntelliJ IDEA Ultimate** (IDE)
- **Jakarta Servlet API 5.0.0**
